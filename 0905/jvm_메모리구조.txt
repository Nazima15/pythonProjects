## 🧠 JVM 메모리 구조 (간단 버전)

```
────────────────────────────
 Method Area (메소드 영역)
 - 클래스 정보
 - 메소드 정보
 - static 변수
────────────────────────────
 Heap (힙)
 - new 로 만든 객체
 - 인스턴스 변수
────────────────────────────
 Stack (스택)
 - 지역 변수
 - 매개 변수
 - 함수 실행 정보
────────────────────────────
```

---

### 📌 비유

* **Method Area** → 📚 수업 교재 (모두가 같이 보는 책)
* **Heap** → 📦 창고 (필요할 때 꺼내 쓰는 물건, 직접 치워야 함)
* **Stack** → 🥞 접시 더미 (위에 쌓았다가 위에서부터 치움)

---

✅ 한 줄 암기:

* **스택**: 함수 실행, 지역 변수 (자동 관리)
* **힙**: 객체 저장 (new, 수동 관리)
* **메소드 영역**: 클래스/메소드/static (공유 공간)
